# Portfolio Sandbox - Exact Copy

This is an exact one-to-one copy of the portfolio website from https://elevate-ux-ai-showcase.lovable.app/

## Setup Instructions

1. Install dependencies:
```bash
npm install
```

If you encounter permission issues with npm cache, run:
```bash
sudo chown -R $(whoami) ~/.npm
```

2. Start the development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

4. Preview production build:
```bash
npm run preview
```

## Project Structure

- `/src` - React components and source code
- `/public` - Static assets (images, etc.)
- `/src/components` - React components (Navigation, Hero, Work, Contact, Footer)
- `/src/index.css` - Global styles and Tailwind imports

## Technologies Used

- React 18
- Vite
- Tailwind CSS
- Google Fonts (Syne, DM Sans)

## Notes

All images have been downloaded from Unsplash and are stored in `/public/images/`. The page structure, styling, and content match the original website exactly.
